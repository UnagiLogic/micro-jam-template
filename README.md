# micro-jam-template
5 day build sessions with the community! Each week there will be a topic. The goal is to make something small that week.

## The Challenge

Each micro-jam presents a unique theme or constraint for participants to work within. You'll have 5 days to create something small but meaningful using the provided template. Whether you're a beginner or experienced developer, the challenge is designed to encourage creativity and rapid prototyping.

## How to Participate

1. **Fork this repository** to create your own copy
2. **Navigate to the `module/` directory** where you'll find the basic template files
3. **Modify the `module.js` file** to implement your creative solution
4. **Test your creation** by opening `index.html` in your browser
5. **Share your work** with the community when you're ready

The template provides a 1000x1000 pixel canvas and basic JavaScript structure to get you started quickly.

## Rules

- **Time limit**: 5 days from the challenge announcement
- **Use the provided template**: Build upon the existing `module/` structure
- **Keep it simple**: The goal is small, focused creations, not large projects
- **Be creative**: Interpret the theme in your own unique way
- **Have fun**: This is about learning, experimenting, and community building
